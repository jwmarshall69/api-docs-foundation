# API Documentation Foundation

This repository contains a reusable Docs-as-Code system for API documentation,
including Markdown docs, OpenAPI specifications, and Postman collections.

## Structure

- /docs – Human-readable documentation
- /openapi – OpenAPI source of truth
- /postman – Request collections for validation
- /templates – Reusable documentation templates

## Audience

API consumers, technical writers, and internal developers.
